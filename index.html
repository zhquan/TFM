Resumen

Q-DashMan es una aplicación web en Django que sirve para generar dashboards de una manera muy simple, usando las herramientas de GrimoireLab.

Actualmente extraer información de las múltiples fuentes de datos de desarrollo de software no es complicado gracias a la herramienta llamado Perceval. Además podemos saber quien está contribuyendo a los proyectos y en que empresa u organización está trabajando, quienes son los que están manteniendo el proyecto vivo, quienes son los que están revisando los códigos, etc. Todo esto es posible gracias a Sortinghat que nos afilia automáticamente según el dominio, nombre de usuario, un fichero, etc. Una vez que haya terminado el proceso de afiliación, tenemos que visualizarlos con Sigils para que podamos entender bien esos datos con gráficas simples y también nos brinda la posibilidad de crear unos nuevos si fuera necesario. Todo esto está orquestado por Mordred para que todas las piezas del puzzle encajen perfectamente y por resultado, un dashboard totalmente personalizado.

Aunque tenemos herramientas para crear un dashboard desde cero. Pero ¿cómo se configuran los ficheros de Mordred para que funcione todo correctamente?
Aquí es donde entra en juego esta aplicación para que el usuario vea un interfaz muy simple y sin tener que escribir código, además sólo rellenando unos simples formularios para que SirModred pueda empezar a hacer su magia para que empiece a recoger los datos de las fuentes, procesarlos hasta la generación de nuestro dashboard.

Por otro lado la configuración de estos ficheros son muy liosos y cada fuente de dato es diferente. Por ejemplo para Git, Mordred necesita saber todos los repositorios git que se van a analizar, además hay que obtener esos repositorios, si están alojados por ejemplo en GitHub nos tenemos que fijar también si son repositorios forks\footnote{Repositorios que son copias de otro o no, ya que normalmente sólo nos interesa saber como están evolucionando nuestros proyectos.
Pues esto en la aplicación con sólo introducir el nombre de la organización de GitHub nos crea directamente el fichero de proyectos en formato de Mordred con todos los repositorios. De igual modo con los canales de Slack y los grupos de Meetup.

En definitiva esta aplicación nos facilita la vida para tener un dashbard personalizable y con todas métricas, visualizaciones, afiliaciones y mucho más. Gracias a la herramienta GrimoireLab desarrollado por Bitergia.
